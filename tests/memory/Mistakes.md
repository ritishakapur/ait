# Mistakes.md

## Purpose
Record of mistakes made during this project. What went wrong and how to avoid it next time.

## Format
- **Date:** [YYYY-MM-DD]
- **Mistake:** [What went wrong]
- **Impact:** [How bad was it?]
- **Root cause:** [Why did it happen?]
- **Fix:** [How we fixed it]
- **Prevention:** [How to avoid it next time]

## Examples
### 2026-07-13
- **Mistake:** Industry.md had duplicate section numbers (two "2.1 Structure" sections)
- **Impact:** Agents would duplicate content or skip sections
- **Root cause:** Template was edited multiple times without renumbering
- **Fix:** Renumbered and merged duplicate sections
- **Prevention:** Always run a section number check before distributing templates to agents
