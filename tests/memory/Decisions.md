# Decisions.md

## Purpose
Record of key decisions made during this project. Why we chose X over Y.

## Format
- **Date:** [YYYY-MM-DD]
- **Decision:** [What was decided]
- **Options considered:** [What else was on the table]
- **Why we chose this:** [Reasoning]
- **Who decided:** [Person/team]
- **Reversible?** [Yes/No — if reversible, note the rollback plan]

## Examples
### 2026-07-13
- **Decision:** Use CSC e-Governance India Ltd as the first company to analyze
- **Options considered:** Jindal Stainless Ltd, another PSU, a private company
- **Why we chose this:** CSC has rich data, clear stakeholder map, and serves as a strong template for future analyses
- **Who decided:** [Team lead]
- **Reversible?** Yes
