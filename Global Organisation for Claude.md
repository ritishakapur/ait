# Global Organisation for Claude.md

## Purpose
This is the master guide for how we work together — you, me, and our colleagues. It defines the rules, workflows, frameworks, and research structure we use to produce strategic analysis.

## How It Works
1. **Rules** → How we think, verify, and write
2. **Workflows** → End-to-end processes for specific tasks
3. **Frameworks** → Mental models and analytical tools
4. **Research** → The output files we produce
5. **Templates** → Reusable structures for new analyses
6. **Tests/Memory** → Decisions, mistakes, and patterns we learn from

## Quick Start
- New to the system? Start with `frameworks/Thinking agent.md` and `frameworks/MECE.md`
- Need to do research? Follow `workflows/Context-Layer.md`
- Need to analyze competition? Follow `workflows/Competitive-Analysis.md`
