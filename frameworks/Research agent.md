# Research agent.md

## Purpose
How to conduct research effectively. This is the playbook for gathering, verifying, and synthesizing information.

## The Research Process

### Phase 1: Scoping
1. Define the research question
2. Identify what we need to know
3. Identify what we already know
4. Identify the gaps

### Phase 2: Source Identification
1. Primary sources first (annual reports, official statements, regulatory filings)
2. Secondary sources next (reputable analysis, industry reports)
3. Tertiary sources last (news articles, summaries)
4. Social sources for ground truth (WhatsApp groups, YouTube tutorials, Reddit threads — for what's implied but not said)

### Phase 3: Evidence Gathering
1. Collect evidence for each claim
2. Label each piece of evidence with: source, date, type (primary/secondary/tertiary)
3. Flag any claim with no evidence
4. Flag any claim with only one source

### Phase 4: Verification
1. Cross-check numbers across sources
2. Verify calculations
3. Check for contradictions between sources
4. Run the CEO test (would the CEO challenge this?)

### Phase 5: Synthesis
1. Organize findings into the research file structure
2. Write in plain language
3. Flag all uncertainties
4. Add Blind Spots section

## Research Rules
- Never present an inference as evidence
- Never present an assumption as evidence
- Always label the source type
- Minimum 2 sources for critical claims
- Minimum 1 source for descriptive claims
- If you can't find a source, write "Estimated: [number] — based on [logic]" and flag it

## What Good Looks Like
- Every claim is backed by evidence
- Every source is cited
- Every estimate is labeled
- Every blind spot is documented
