# Evidence.md

## Rule: What Counts as Evidence
Every claim must be backed by evidence. Not opinions. Not assumptions. Evidence.

## Evidence Hierarchy (Strongest to Weakest)
1. **Primary source** — Original document, annual report, official statement, direct quote
2. **Secondary source** — Reputable analysis that cites primary sources
3. **Tertiary source** — News articles, summaries, recaps
4. **Inference** — Logical deduction from known facts (flag as "inference")
5. **Assumption** — What we believe to be true but cannot prove (flag as "assumption")

## Rules
- Never present an inference as evidence
- Never present an assumption as evidence
- Always label the source type
- If a claim has no evidence, flag it in Blind Spots
- Minimum 2 sources for critical claims (revenue, market share, strategy)
- Minimum 1 source for descriptive claims (product features, service listings)

## What Good Looks Like
- "CSC processed 373 lakh transactions per month (Source: CSC Annual Report 2024, verified by MeitY public data)"
- Not: "CSC processes a lot of transactions"
